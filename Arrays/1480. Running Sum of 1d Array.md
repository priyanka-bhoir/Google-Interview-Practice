# 1480. Running Sum of 1d Array

``` Problem Statement
Given an array nums. We define a running sum of an array as runningSum[i] = sum(nums[0]â€¦nums[i]).

Return the running sum of nums.

 ```

## Soluion:
 class Solution {
    public int[] runningSum(int[] nums) {
     
        int i= 1;
        while(i<nums.length){
            nums[i]+=nums[i-1];
            i++;
        }
        return nums;
        
    }
}

## Complexities:

1. Time =O(n)
2. Space O(1)
